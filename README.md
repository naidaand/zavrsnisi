# Salon App

Web aplikacija za rezervaciju termina u frizerskom salonu.

## Funkcionalnosti

* Registracija i login (Firebase)
* Dodavanje rezervacija
* Pregled rezervacija
* Brisanje rezervacija

## Tehnologije

* HTML, CSS, JavaScript
* Firebase Authentication
* PHP + MySQL (XAMPP)

## Pokretanje

1. Pokreni XAMPP (Apache + MySQL)
2. Otvori:
   http://localhost/salon_app/login.html

## Napomena

Korisnici se čuvaju u Firebase, a rezervacije u MySQL bazi.
